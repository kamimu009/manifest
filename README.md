# manifest

1. install repo

    ```shell
    curl https://storage.googleapis.com/git-repo-downloads/repo > ~/bin/repo
    chmod a+x ~/bin/repo
    ```
    > https://source.android.com/setup/build/downloading  
1. create dhirectory

    ```shell
    $ mkdir rise-app
    $ cd rise-app
    ```
1. repo init

    ```shell
    $ repo init -u git@github.com:kamimu009/manifest.git
    ```
1. repo sync

    ```shell
    $ repo sync
    ```

Done!
